# Rubik's Cube — Lessons

A guided, beginner-to-literacy course on solving the 3×3 Rubik's Cube. Each lesson is a
self-contained, interactive HTML page — built to teach *why* each move works, not just the steps.

## 📱 Open the lessons

Live site (works on phone): **see the GitHub Pages link in the repo's About section**, or open
[`index.html`](index.html).
